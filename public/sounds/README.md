# Arquivos de Som Necessários

Para que os efeitos sonoros funcionem corretamente na Arena de Vendas, por favor adicione os seguintes arquivos nesta pasta (`public/sounds`):

1. **`horn.mp3`**: Som de corneta para novas vendas, combos e metas batidas.
   - Sugestão: Um som curto e vibrante de corneta de estádio ou festa.

2. **`race-car.mp3`**: Som de carro de corrida acelerando para ultrapassagens e liderança.
   - Sugestão: Um som de motor V8/F1 passando rapidamente (efeito Doppler).

> **Nota:** Se os arquivos não existirem, o sistema tentará reproduzir mas falhará silenciosamente (apenas erro no console), sem quebrar a aplicação.
